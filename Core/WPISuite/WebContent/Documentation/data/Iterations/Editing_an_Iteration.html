Editing a Iteration

To edit an iteration:

  • Click "Requirements Manager" a tab at the top
  • Select an iteration you wish to edit in the requirements tree view, located on the left side.
  • Right click and select "Edit Iteration"
  • Edit the name and/or date fields to reflect the updated information, making sure that all requirements of an iteration are still met
  • Click the save button at the bottom of the page

    • A progress bar will be displayed with the current completion percentage for the iteration.
