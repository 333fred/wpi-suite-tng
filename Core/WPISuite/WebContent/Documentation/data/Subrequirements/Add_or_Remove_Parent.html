Add or Remove Parent from a Requirement

Adding a parent to a requirement means that it depends on its parent's completion in order to be completed itself. Requirements may have one parent.

Adding Parent

  • Open a requirement that will a parent
  • Navigate to the "Subrequirements" tab in the right panel of the requirement edit view
  • There is a list of existing requirements in the "Add to SubRequirement Hierarchy" section of this panel. Make sure the "Add Parent" radio button is selected and then choose a requirement that you wish to add as a parent so that it is highlighted in blue
  • Click the "Assign Parent" button to add the parent
  • The parent will be added next to the "Parent:" label above

    • Click image to enlarge

    A parent has been added

  • If you open the parent, it will have the children assigned to it in the "Children:" list under the Subrequirements tab

    • Parents may not be deleted if they contain children.


Removing Parent

  • Open a requirement that you want to remove an existing parent from
  • Navigate to the "Subrequirements" tab in the right panel of the requirement edit view
  • Click the "Remove Parent" button to remove the child
  • The parent will no longer be displayed next the the parent label

    • Click image to enlarge

    Create Iterations has been removed as a parent


Changing Parent

  • Open a requirement that you want to change an existing parent
  • Navigate to the "Subrequirements" tab in the right panel of the requirement edit view
  • Click the "Add parent" radio button and select the requirement you want to change the parent to
  • click the "Change Parent" button to change the parent
