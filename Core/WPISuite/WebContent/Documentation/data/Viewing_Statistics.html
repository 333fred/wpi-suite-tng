Viewing the Statistics of the Database

To view the statistics of the requirements database:

  • Click the 'Requirements Manager' tab at the top of the screen
  • The default chart you will see is a pie chart of all of the requirement's status
  • You may change the statistic type by using the drop down menu on the right with the label "Statistic Type:"
  • You also have to option to switch which type of chart to view by clicking one of the radio buttons on the right
    • Pie Chart: view a pie chart with the statistics for the requirement database
    • Bar Chart: view a bar chart with the statistics for the requirement database
    • Line Chart: view a line chart with the statistics for the requirement database
