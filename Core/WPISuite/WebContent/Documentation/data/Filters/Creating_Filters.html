Creating Filters

Navigate to the filters by clicking on the "Filters" tab in the requirement tree view in the left most panel of the Janeway client.

You can filter out requirements in the table view by any of it's fields, which includes:
   Name, Type, Priority, Status, Iteration, Estimate, Effort, and Release Number

Each field has unique filter options that can be used to separate requirements. The filters will require you to use dropdown menus that contain elements for each field or type in values that will be used for filtering purposes. Each filter contains a Field, and Operation, and a Value Here is a list of each Operation and what each one means:

   Equal: The value for the filter will be equal to to value for the field specified.
   Not Equal: Requirements with fields that do not equal the value of the filter will be separated and displayed.
   Starts With: Requirements with a field that start with the value will be filtered.
   Contains: Requirements with a field that contain the value will be filtered. The value can be anywhere in the specified field.
   Occurs Before: Requirements in iterations that occur before a specific date will be filtered.
   Occurs After: Requirements in iterations that occur after a specific date will be filtered.
   Occurs Between: Requirements in iterations that occur between two specific dates will be filtered.

A small button located on the right side of the text box for the operations Occurs Before, Occurs After, and Occurs Between will be available that will display a calendar like the one seen in "Creating an Iteration" to choose dates.

    <: Requirements with a field that is less than the value will be filtered.
    <= : Requirements with a field that is less than or equal to the value will be filtered.
    >: Requirements with a field that is greater than the value will be filtered.
    >= : Requirements with a field that is greater than or equal to the value will be filtered.

After you have completed filling out the fields for your filter, click the "Save" button on the bottom of the filter tab panel. Your filter will appear in the space above the field selection.
