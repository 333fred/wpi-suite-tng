Adding Users to a Project

Here is how add users to a project.

  • log in using your admin username and password.
  • Under "Team Members" click on the Project idNum text box and type an existing projects ID number (refer to CREATE PROJECT L INK HERE for instructions on how to create a project)
  • Now click on the Username text box and type the username of the user you wish to add to the project
  • Click the "Add Team Member" button to add that member to the project. The user will now be able to connect to Janeway under that project name
