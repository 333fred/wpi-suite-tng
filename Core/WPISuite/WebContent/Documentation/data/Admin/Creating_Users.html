Creating a User as ADMIN

Here is how to create an account and a password for a user using your admin credentials.
Navigate to the WPI Suite Admin Console using this link:

  • log in using your admin username and password.
  • Under "Core User" click on the Username text box and type a username
  • Assign that user a name using the Name text box
  • Give that user a password that they will have access to or will remember using the Password text box
  • Each user will have a unique ID number they will use to get assigned to a project. Give them an ID number using the ID text box
  • Press the "Create CoreUser" button
