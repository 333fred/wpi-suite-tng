Creating a Project

Here is how to create a project.

  • log in using your admin username and password.
  • Under "CoreProject" click on the Project idNum text box and type an unique project ID
  • Now click on the Name text box and type the name of your new project
  • Click the "Create Project" button to add project. You may now add users to that project. Refer to for more details on how to add users to projects
