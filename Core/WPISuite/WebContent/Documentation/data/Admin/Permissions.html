Permissions

Permissions determine what rights a user has and what they can do. Only administrators may edit permissions.

In order to edit permissions you must be assigned as an administrator on the current project.
The permission of the current user, as well as the 'admin' user can't be edited.
New users that are added to the project will have the Observer permission.
