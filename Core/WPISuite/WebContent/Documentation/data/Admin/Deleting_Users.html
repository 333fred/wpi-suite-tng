Creating a User as ADMIN

Here is how to delete an account using your admin credentials.
Navigate to the WPI Suite Admin Console using this link:

  • log in using your admin username and password.
  • Under "Core User" click on the Username text box and type an username you wish to delete
  • Type that user's name using the Name text box
  • Type that user's password using the Password text box
  • Type that user's ID using the ID text box
  • Press the "Delete CoreUser" button
