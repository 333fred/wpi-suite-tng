Deleting Projects

Here is how to delete an existing project using your admin credentials.
Navigate to the WPI Suite Admin Console using this link:

  • log in using your admin username and password.
  • Under "CoreProject" click on the Project idNum text box and type the ID number of the project you wish to delete
  • Type the project's name in the name text box
  • Press the "Delete CoreProject" button
