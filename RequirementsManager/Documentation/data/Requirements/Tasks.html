Tasks

Tasks are small, divisible portions of a requirement. To add new tasks or edit existing tasks to an existing tasks to a requiremnt, follow these steps:

  • Edit the requirement and select the "Tasks" tab from the panel on the right.
    • To add a new task, begin typing in the name and description fields
    • To edit an existing task, select the task from the list of tasks.
  • Enter the name and description into the appropriate fields.
  • Each task can be marked complete independently. To do this, simply check the box labeled "Completed."
  • A task can have a single user assigned to it. To assign a user, select the user from the dropdown menu labled "Users."
